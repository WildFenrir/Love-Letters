# 🎨 Система Тем

Система персонализации внешнего вида приложения "Секретные Записки".

## 📋 Возможности

### ✅ Реализовано
- **5 тем на выбор**:
  - 💜 **Классическая** - фиолетовая гамма (по умолчанию)
  - 🌿 **Природа** - зелёная, свежесть
  - 🌊 **Океан** - синяя, спокойствие
  - 🌙 **Тёмная** - ночной режим
  - 🌅 **Закат** - оранжевая, тепло

- **Разные темы для каждого пользователя**:
  - Админ выбирает свою тему
  - Получатель выбирает свою тему
  - Темы не зависят друг от друга

- **Админ видит тему получателя**:
  - В админке есть секция "Тема получателя"
  - Показывает текущую тему и дату обновления

- **Красивый UI**:
  - Модальное окно с предпросмотром
  - Анимация открытия/закрытия
  - Кнопка 🎨 в правом нижнем углу
  - Плавные переходы между темами

### 🔮 В будущем (можно добавить)

- Кастомные фоны (загрузка пользователем)
- Анимации переходов между разделами
- Сезонные темы (Новый год, Хэллоуин и т.д.)
- Экспорт/импорт настроек темы

## 🚀 Установка

### 1. Выполнить SQL скрипт

```sql
-- В Supabase SQL Editor выполнить:
setup-themes.sql
```

### 2. Подключить CSS

```html
<!-- В index.html и admin.html -->
<link rel="stylesheet" href="themes.css">
```

### 3. Использовать темы по умолчанию

Тема применяется автоматически при загрузке страницы.

## 📁 Файлы

| Файл | Описание |
|------|----------|
| `themes.css` | Стили для 5 тем и модального окна |
| `setup-themes.sql` | SQL скрипт для таблицы тем |
| `THEMES.md` | Эта документация |

## 🎨 Структура темы

Каждая тема определяет CSS переменные:

```css
:root[data-theme="nature"] {
  --bg-primary: #f0fdf4;      /* Основной фон */
  --bg-card: #ffffff;          /* Фон карточек */
  --text-primary: #14331f;     /* Основной текст */
  --accent-primary: #16a34a;   /* Акцентный цвет */
  --gradient-primary: ...      /* Градиент кнопок */
  --bg-pattern: url(...);      /* Паттерн фона */
}
```

## 💡 Как это работает

### Для получателя:
1. Нажимает кнопку 🎨 в правом нижнем углу
2. Выбирает тему из 5 вариантов
3. Тема сохраняется в базу (если авторизован) или localStorage
4. Тема применяется мгновенно

### Для админа:
1. Тоже самое для выбора своей темы
2. Дополнительно видит тему получателя в секции "Тема получателя"

## 🔐 Хранение

| Место | Что хранится |
|-------|--------------|
| **Supabase `themes` таблица** | Тема авторизованного пользователя |
| **localStorage** | Тема для гостя/фолбэк |

## 🎯 Примеры использования

### Добавить новую тему

1. **Добавить в `themes.css`**:
```css
:root[data-theme="purple"] {
  --bg-primary: #faf5ff;
  --accent-primary: #9333ea;
  /* ... остальные переменные */
}
```

2. **Добавить в HTML** (в модальное окно):
```html
<div class="theme-option" data-theme="purple">
  <div class="theme-option-preview purple"></div>
  <div class="theme-option-name">Фиолетовая</div>
  <div class="theme-option-desc">Яркая тема</div>
  <div class="theme-checkmark">✓</div>
</div>
```

3. **Добавить стиль предпросмотра**:
```css
.theme-option-preview.purple {
  background: linear-gradient(135deg, #faf5ff, #f3e8ff);
}
```

## 🐛 Известные ограничения

- Паттерны фона могут незначительно влиять на производительность
- При переключении темы все элементы меняют цвет плавно (0.3s)
- Для неавторизованных пользователей тема хранится только в браузере
